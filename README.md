# Homelab Journey

This repository documents my hands-on learning journey into Linux, virtualization, networking, self-hosting, and systems administration.

The goal is not only to build useful services, but also to understand the underlying concepts and document the challenges, mistakes, troubleshooting processes, and solutions encountered along the way.

## Current Topics

* Git & GitHub
* Proxmox VE
* Linux Systems Administration
* Storage and Filesystems
* LXC Containers
* Nextcloud
* Networking
* Remote Access
* Backup Strategies
* Self-Hosted Services

## Objectives

* Learn Linux and systems administration through hands-on projects.
* Build and maintain a self-hosted infrastructure.
* Document challenges, troubleshooting steps, and solutions.
* Develop practical skills relevant to IT, infrastructure, and system administration roles.
* Create a public record of projects, lessons learned, and technical growth.

## Philosophy

This repository focuses on learning by building real services rather than following tutorials blindly.

Every project aims to document:

* Goals
* Architecture decisions
* Commands used
* Problems encountered
* Root cause analysis
* Troubleshooting process
* Final solution
* Lessons learned

The objective is to understand not only what works, but why it works.
